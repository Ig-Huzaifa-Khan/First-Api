# My First API

A simple Express.js API project.

## Description
This is a basic Node.js API built with Express.js framework.

## Installation

1. Clone the repository
```bash
git clone <repository-url>
cd my-first-api
```

2. Install dependencies
```bash
npm install
```

## Usage

Run the server:
```bash
node server.js
```

## Dependencies
- Express.js ^5.1.0
