# First-Api

A simple Express.js API project.

## Description
This is a basic Node.js API built with Express.js framework.

## Installation

1. Clone the repository
```bash
git clone https://github.com/Ig-Huzaifa-Khan/First-Api.git
cd First-Api
```

2. Install dependencies
```bash
npm install
```

## Usage

Run the server:
```bash
node server.js
```

## Dependencies
- Express.js ^5.1.0
